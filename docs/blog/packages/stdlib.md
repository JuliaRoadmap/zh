# 标准库
[中文文档](https://docs.juliacn.com/latest/)侧边栏 `Standard Library` 处可以找到当前标准库找到列表。
部分标准库是位于 `Base` 下的子[模块](../../advanced/module.md)，部分标准库可能需要安装。
