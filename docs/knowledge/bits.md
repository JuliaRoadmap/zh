# 位与位运算
## 位
众所周知，在常见的计算机结构中，数据均以「位(bit)」存储。为了便于管理，每 8 位组成 1 个「字节(byte)」。使用前置修饰符，如 `K` —— 组成 `KB` 表示 $2^10$ byte 等。在硬件中，数据存储可能存在分区、分块等措施，此处不进行阐述。

利用 `bitstring` 函数，你可以方便的列出数字的各位数据

## 无符号整数
无符号整数的存储方式是最容易理解的：即二进制数。
以 $22=(10110)_2$ 为例：
| 单位数值 | 权值 | 结果 |
| --- | --- | --- |
| 0 | 1 | 0 |
| 1 | 2 | 2 |
| 1 | 4 | 4 |
| 0 | 8 | 0 |
| 1 | 16 | 16 |
