# github 使用基础
## 关于
Github 是当前使用最广泛的 [git](git.md) 平台。

Github 主站：<https://github.com>

## 账户
按照网站提示即可完成注册。

您可以直接使用 `/login`、`/logout` 路径登录登出。应特别注意账号安全，因为很多平台可能都支持与 github 账户进行绑定。
