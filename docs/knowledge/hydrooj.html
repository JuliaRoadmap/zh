    HydroOJ使用

    HydroOJ是一个基于Hydro，支持Julia语言的在线评测器(OJ,Online Judge)

    官方使用指南

    在递交时，应将代码包裹进函数中，以方便优化

    function main()
	# 代码
end
main()

    Note

    由于Julia的JIT需要启动时间，管理员设置了二倍时限

    安全

    Hydro主要开发者undefined-moe较重视安全问题

    相关事项请信任超级管理员(Super User)（带有SU标签），可选择性信任携带其它标签者
