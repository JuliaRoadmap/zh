## LaTeX
LaTeX 是一种渲染文档、数学公式所用的表达语言（当然，一些 Tex 宏包还支持有机化学等）。
通常使用的是它的一部分，[KaTeX](https://katex.org/)，主要用于表达数学公式。
在 julia-markdown 中，它的嵌入可以采用传统的美元，即一对 `$ $`，也可以使用一对 ``` `` `` ```
