# 许可证知识
## 关于
对于每个开源项目，应该提供一个许可证（不提供并让别人问也是一种选择，但不提倡）。

Github上使用率最高的五个许可证是MIT、Apache 2.0、GPL 2.0、GPL 3.0和bsd-3-clause[^1]，而文本通常使用[CC族协议](https://creativecommons.org/licenses/)
