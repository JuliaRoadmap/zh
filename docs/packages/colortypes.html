    ColorTypes的使用

    ColorTypes引入了一系列色彩相关的类型，底层使用了FixedPointNumbers包

    julia> c=RGB(0.5,0.5,0.5)
RGB{Float64}(0.5,0.5,0.5)

julia> red(c) # 或 c.r
0.5

julia> c=RGBA(1,1,0,0)
RGBA{N0f8}(1.0,1.0,0.0,0.0)

julia> alpha(c)
0.0N0f8

julia> alpha(c).i
0x00
