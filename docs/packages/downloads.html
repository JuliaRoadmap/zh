    Downloads的使用

    Downloads是一个标准库，基于Curl，用于满足基本的爬虫需求

    Downloads支持的各项设置较少，如需更多设置请使用HTTP包

    request

    原型：

    request(url;
	[ input = , ]
	[ output = , ]
	[ method = input ? "PUT" : output ? "GET" : "HEAD", ]
	[ headers = , ]
	[ timeout = , ]
	[ progress = , ]
	[ verbose = false, ]
	[ throw = true, ]
	[ downloader = , ]
) -> Union{Response, RequestError}


    julia> io=IOBuffer()
IOBuffer(data=UInt8[...], readable=true, writable=true, seekable=true, append=false, size=0, maxsize=Inf, ptr=1, mark=-1)

julia> Downloads.request("http://httpbin.org/user-agent";output=io)
Response("http", "http://httpbin.org/user-agent", 200
...

julia> String(take!(io))

    download

    建议将下载作为它的唯一功能，其它功能用request实现

    julia> fio=open("D:/foo.png","w")
IOStream()

julia> Downloads.download("http://httpbin.org/image/png",fio);
