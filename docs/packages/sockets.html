    Sockets的使用

    Sockets提供了网络套接字

    一个在localhost:2333显示HTML的实例：

    str="""
HTTP/1.1 200 OK
Content-Type: text/html;charset=UTF-8
	

	
		
    TEXT
    
	

""" # 按照标准HTTP Response格式

@async begin
	server=listen(2333)
	while true
		sock=accept(server)
		print(sock,str)
	end
end
