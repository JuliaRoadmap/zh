# 延伸阅读
本页汇总了 [Modern Julia Workflows - Going further](https://modernjuliaworkflows.org/further/) 页面提及的深入学习 Julia 的优质外部资源。[^1]

## 官方资源
- [Julia 官方文档](https://docs.julialang.org/en/v1/)
- [Julia 中文文档](https://docs.juliacn.com/latest/)
- [Julia 官方博客](https://julialang.org/blog/)
- [JuliaHub](https://juliahub.com/)：包搜索、在线运行与协作平台
- [Julia Discourse 论坛](https://discourse.julialang.org/)
- [Julia 中文论坛](https://discourse.juliacn.com/)

## 教程
- [Julia Academy](https://juliaacademy.com/)：官方合作的免费视频课程
- [Introduction to Computational Thinking](https://computationalthinking.mit.edu/)（MIT 18.S191）：基于 Pluto.jl 的互动课程，涵盖图像处理、气候科学等
- [Julia By Example](https://juliabyexample.helpmanual.io/)：以示例驱动的快速参考
- [Think Julia](https://benlauwens.github.io/ThinkJulia.jl/latest/book.html)：面向初学者的免费电子书
- [Statistics with Julia](https://statisticsjulia.org/)：面向统计与数据科学的 Julia 入门

## 博客与深度文章
- [Forem Julia 社区](https://forem.julialang.org/)：社区成员发布的技术文章
- [Bogumił Kamiński 的博客](https://bkamins.github.io/)：Julia 数据科学与 DataFrames.jl 专题
- [Chris Rackauckas 的博客](https://www.stochasticlifestyle.com/)：Julia 性能、微分方程与科学计算
- [Julia Folds2 组织博客](https://juliafolds2.github.io/)：并行计算与函数式编程

## 视频
- [JuliaCon 历届视频](https://www.youtube.com/@TheJuliaLanguage)：Julia 年度大会的演讲录像
- [The Unreasonable Effectiveness of Multiple Dispatch](https://www.youtube.com/live/kc9HwsxE1OY?feature=share)：Stefan Karpinski 在 JuliaCon 2019 的演讲
- [Julia for Pythonistas](https://www.youtube.com/watch?v=4Q1sYdGm0dY)：面向 Python 用户的 Julia 入门

## 历史与背景（Lore）
以下文章记录了 Julia 语言的发展历程，有助于理解其设计哲学：

- [Why we created Julia](https://julialang.org/blog/2012/02/why-we-created-julia/)（2012）：Julia 诞生的初衷
- [Paper in SIAM Review: Julia - A Fresh Approach to Numerical Computing](https://julialang.org/blog/2017/03/julia-fresh-paper/)（2017）：Julia 核心论文
- [Announcing the release of Julia 1.0](https://julialang.org/blog/2018/08/one-point-zero/)（2018）：Julia 1.0 正式发布公告
- [The Julia Project and Its Entities](https://julialang.org/blog/2019/02/julia-entities/)（2019）：Julia 项目的组织结构
- [Julia's Release Process](https://julialang.org/blog/2019/08/release-process/)（2019）：版本发布流程说明
- [Code, docs, and tests: what's in the General registry?](https://julialang.org/blog/2021/08/general-survey/)（2021）：对 General 注册表的调查分析
- [Why We Use Julia, 10 Years Later](https://julialang.org/blog/2022/02/10years/)（2022）：十年后重谈使用 Julia 的理由
- [Julia 1.9 Highlights](https://julialang.org/blog/2023/04/julia-1.9-highlights/)（2023）：Julia 1.9 新特性亮点
